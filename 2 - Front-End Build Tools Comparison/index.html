A Comparison of Front-End Build Tools

  1. Assemble

    Grunt templating generator. I like this one. Less automated than the others, but because of it I was able to understand what was going on much better.

  2. JekyllRB

    A blog generator, essentially. It's a generator for Yeoman using Grunt and Bower, but it's too encapsulated for me to figure out how the hell it works. Its documentation said it uses Liquid markup. I probably won't be using this one.

  3. Yo Webapp

    The most basic Grunt build tool I used, and I could see myself using it for simple pages that didn't require templating or using partials.

  4. Web Fundamentals

    Google's answer to the front-end build tools. It's essentially the same as the yo webapp generator, but it uses Gulp instead of Grunt. While Gulp does seem to be a bit more simple, it's newer and doesn't have as many well-created packages for it.
